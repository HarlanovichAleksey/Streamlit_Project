# 📊 Анализ заработных плат в России с учётом инфляции

## 📌 Описание проекта

Данное Streamlit-приложение предназначено для анализа динамики средних заработных плат в Российской Федерации по видам экономической деятельности за период 2000-2025.

В проекте проводится сравнение **номинальной** и **реальной (с учётом инфляции)** заработной платы, а также анализируются темпы роста, структура распределения зарплат и влияние инфляции на доходы населения.

---

## 🎯 Цель проекта

* Проанализировать динамику заработных плат по отраслям экономики
* Сравнить номинальные и реальные значения зарплат
* Оценить влияние инфляции на уровень доходов
* Выявить отрасли-лидеры по росту заработной платы
* Визуализировать экономические тенденции с помощью интерактивного дашборда

---

## 📂 Используемые данные

### 1. Заработные платы

Источник: Росстат
Ссылка: [https://rosstat.gov.ru/labor_market_employment_salaries](https://rosstat.gov.ru/labor_market_employment_salaries)

* Среднемесячная номинальная начисленная заработная плата
* Разрез по видам экономической деятельности
* Период: 2000-2025

Источник: открытые данные по инфляции РФ

## ⚙️ Функционал приложения

Приложение реализует следующие возможности:

* 📈 Построение динамики заработных плат (номинальных и реальных)
* 🔥 Тепловые карты изменения зарплат по отраслям
* 📊 Анализ темпов роста заработной платы
* 📦 Распределение зарплат по отраслям
* 🚀 Лидеры роста зарплат (номинальный и реальный рост)
* 📌 Итоговые метрики (средняя, медианная, минимальная, максимальная зарплата)
* 💡 Учёт инфляции при расчётах реальных доходов

---

## 🧠 Основные выводы

* Номинальная заработная плата стабильно растёт, однако реальный рост значительно ниже из-за инфляции
* В отдельных периодах наблюдаются отрасли, где рост зарплат опережает инфляцию
* Реальная покупательная способность доходов изменяется менее выраженно, чем номинальные показатели
* Существуют заметные различия в динамике между отраслями экономики
